Welcome to the Movie API

This API provides access to information about different movies, directors, and genres. You can use the provided endpoints to perform various operations.

Getting Started

To get started, you can:

  • View the API Documentation to learn about available endpoints.
  • Interact with the API using tools like Postman or directly from your application.

Endpoints

Here are some of the main endpoints you can use:

  • GET /movies: Retrieve information about all movies.
  • GET /movies/:title: Retrieve information about a specific movie.
  • POST /users/register: Register a new user.

About

This API is built with Node.js and Express.js. It uses data storage and supports basic CRUD operations.
