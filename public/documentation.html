REST API Documentation

This application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Operation Request URL Response
Get all movies GET /movies JSON array
Get a single movie GET /movies/:title JSON object
Get data about a genre GET /genres/:name JSON object
Get data about a director GET /directors/:name JSON object
User Registration POST /users/register JSON object
Update user info PUT /users/:userId JSON object
Add movie to favorites POST /users/:userId/favorites Text confirmation
Remove movie from favorites DELETE /users/:userId/favorites/:movieId Text confirmation
User deregistration DELETE /users/:userId Text confirmation
