REST API Documentation

This application provides access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Operation Request URL Response Query Parameters Request Body
Get all movies GET /movies JSON array None N/A
Get a single movie GET /movies/:title JSON object :title N/A
User Registration POST /users/register JSON object N/A User data JSON
Update user info PUT /users/:userId JSON object :userId User data JSON
Add movie to favorites POST /users/:userId/favorites Text confirmation :userId Movie ID JSON
Remove movie from favorites DELETE /users/:userId/favorites/:movieId Text confirmation :userId, :movieId N/A
User deregistration DELETE /users/:userId Text confirmation :userId N/A

For more detailed information, please refer to the API documentation.
